Objectif

Vous allez jouer à un jeu qui possède une stratégie gagnante et contre un joueur qui connait cette stratégie. Le but du jeu est de programmer cette stratégie pour gagner. Pour vous aider, vous pouvez regarder le théorème de Sprague-Grundy.

  Règles

Voici les règles du jeu : Le jeu se déroule sur une grille.
Sur chaque ligne, chaque joueur possède un jeton de sa couleur qu'il peut déplacer uniquement vers le jeton adverse de cette ligne (il ne peut pas reculer) sans pouvoir le dépasser. Autrement dit, une fois que les deux jetons sont l'un à coté de l'autre, plus aucun mouvement n'est possible sur cette ligne.
Chaque joueur déplace un pion à tour de rôle.
Le gagnant est le joueur qui arrive à bloquer totalement son adversaire. Autrement dit, le perdant est le premier joueur qui ne peut plus jouer aucun coup.
C'est toujours vous qui commencez.

Conditions de victoire
  • Avoir réussi à bloquer totalement son adversaire.
Conditions de défaite
  • Ne plus pouvoir jouer aucun coup.
  • Ne pas répondre assez rapidement ou bien donner une réponse non valide.

  Entrées du jeu

Entrées initiales
Ligne 1 : Le nombre rows de lignes.
Ligne 2 : Le nombre columns de colonnes.
Entrées pour un tour de jeu
Pour chaque tour de jeu, on donne dans une boucle pour chaque ligne :
La position xPlayer de votre pion suivie de la position xBoss du pion adverse.
La numeration commence à 0.
Sortie pour un tour de jeu
Ligne 1: 2 entiers séparés par un espace row et newX désignant la ligne du pion qu'on souhaite déplacer suivi de la case sur laquelle on souhaite le déplacer.
Contraintes
1 ≤ rows ≤ 40
3 ≤ columns ≤ 35
0 ≤ xPlayer < xBoss ≤ columns-1
Temps de réponse par tour ≤ 50ms
